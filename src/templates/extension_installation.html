{% extends 'base1.html' %} {% block modal_title %} Credible Web Extension {% endblock modal_title %} {% block modal_body %} We are using frames to embed a web page so that you can evaluate it inside our website. However, some websites(famous ones including Google) don't support this for privacy and security issues. To make up for this, we have developed an extension(currently only supports Chrome browser) with which you can evaluate any page while browsing it. You can download the extension here, and follow these steps to add the extension to Chrome:
  • Download the extension file and save it to your computer.
  • Click the wrench icon on the browser toolbar.
  • Select Tools > Extensions.
  • Locate the extension file on your computer and drag the file onto the Extensions page.
  • Review the list of permissions in the dialog that appears. If you would like to proceed, click Install.
    • (Reference From Google: Adding extensions from other websites)
